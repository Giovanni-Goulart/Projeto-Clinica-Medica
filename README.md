# 🏥 Clínica Médica Life

Este é um projeto de um site institucional para uma clínica médica multiespecialidade. O objetivo principal é apresentar os serviços, diferenciais, especialidades e informações de contato de forma clara e profissional.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização visual, layout com Flexbox e responsividade.
* **Google Fonts**: Utilização da fonte *Source Sans Pro* para uma tipografia moderna.
* **Git & GitHub**: Controle de versão e hospedagem do código.

## 📱 Funcionalidades

* **Página Inicial (Home)**: Visão geral da clínica com banner principal.
* **Seção Sobre**: Detalhes sobre a história, valores e diferenciais da clínica.
* **Especialidades**: Listagem das áreas de atuação (Ortopedia, Pediatria, Geriatria, etc.) com cards estilizados.
* **Contato**: Informações de horário de funcionamento, endereço com mapa integrado e link para agendamento.
* **Responsividade**: O site é adaptável para dispositivos móveis (tablets e smartphones).
